# Programación Orientada a Objetos

Repositorio de práctica para aprender conceptos de programación orientada a objetos.

## Contenido del repositorio

Este repositorio incluye diferentes ejercicios en C++:

### Ejercicio 1
Juego **Piedra, Papel o Tijera** contra la computadora.

Características:
- Generación de jugadas aleatorias
- Comparación de resultados
- Determinación de ganador

### Ejercicio 2
Versión mejorada del juego con:
- Funciones separadas
- Varias rondas
- Sistema de puntuación

### Ejercicio 3
Juego **Mini-Cacho** con dados.

Características:
- Evaluación de tríos
- Escaleras
- Suma de puntos
- Marcador por rondas

## Lenguaje utilizado
